# ehr-app

An Electron application based on EHR data displaying daily working hours.

## Feature

-  The app calculates the end time of a workday based on the start time and displays a countdown in the system tray.
- The app automatically logs in to a system or website based on pre-configured settings.

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
